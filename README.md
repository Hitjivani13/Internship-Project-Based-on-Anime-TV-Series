# Anime & TV Tracker — Demo

This is a small demo prototype for an Anime & TV Series Tracker built with Express + SQLite and a minimal frontend.

Quick start

1. Install dependencies

```bash
npm install
```

2. Start the server

```bash
npm start
```

3. Open http://localhost:3000/

Notes
- The demo uses SQLite (data/db.sqlite). Sample data is seeded automatically.
- API endpoints live under `/api/*` (see `server.js`).
- This is a simplified prototype: add features (clubs, spoilers, polls, analytics) as next steps.
